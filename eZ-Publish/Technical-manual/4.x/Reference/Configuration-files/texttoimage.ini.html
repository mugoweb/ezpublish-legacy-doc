texttoimage.ini

[DefaultSettings]

SettingSummary
Angle Sets the default font angle.
BackgroundColor Sets the default background color for text.
Family This setting set the default font family to use
HeightAdjustment Sets height of the text.
PointSize Sets the default font size.
TextColor Sets the default text color.
WidthAdjustment Sets width of the text.
XAdjustment Sets the left/right placement of the text.
YAdjustment Sets the top/bottom placement of the text.

[ImageSettings]

SettingSummary
UseCache Enables or disables the use of cache files for text transformation.

[PathSettings]

SettingSummary
CacheDir Sets the location of the folder that contains the relevant cache files.
FontDir Sets the location of the folder that contains the fonts.
HtmlDir Sets the location of the HTML folder that contains the cache files of the text to image.

[{style}]

SettingSummary
Angle Sets a specific font angle for the style defined by the configuration block.
BackgroundColor Sets the background color for text for the specific style defined by the configuration block.
Family Sets a specific font family for the style defined by the configuration block.
HeightAdjustment Adjust the height of the text for the specific style defined by the configuration block.
PointSize This setting defines the point size for the specific style defined by the configuration block. Usage
TextColor Sets the text color for the specific style defined by the configuration block.
WidthAdjuastment Adjusts the width the text for the specific style defined by the configuration block.
XAdjustment Adjusts the left/right placement of the text for the specific style defined by the configuration block.
YAdjustment Adjusts the top/bottom placement of the text for the specific style defined by the configuration block.
