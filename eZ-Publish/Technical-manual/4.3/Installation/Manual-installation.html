Manual installation

This installation method is for advanced users who know what they are doing, all other users should use the "Normal installation method". The manual installation method requires an environment which already has a web server, a database and etc. setup and ready to go; eZ Publish needs to be downloaded and unpacked. Instead of running the setup wizard, all configuration is done manually using the command line interface of the target operating system. The following sections (depending on the target OS) will take you through the necessary steps.
