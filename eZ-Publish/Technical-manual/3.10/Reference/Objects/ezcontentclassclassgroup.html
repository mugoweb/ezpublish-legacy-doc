ezcontentclassclassgroup

Summary

Contains information about a class group assignment.

Persistent

Yes.

Attributes

Attribute

Type

Static*

Description

contentclass_id

string

Yes.

The ID number of the class which belongs to the group.

contentclass_version

string

Yes.

The version (either 1 or 0).

group_id

string

Yes.

The ID number of the class.

group_name

string

Yes.

The name of the class group (for example "Media").

* Indicates whether the attribute provides a pre-fetched value or if it needs to be computed upon request.
