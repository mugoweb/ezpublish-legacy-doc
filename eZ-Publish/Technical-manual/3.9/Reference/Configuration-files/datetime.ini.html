    In this file, you can write class format date :
    [ClassSettings]
    Formats[]
    Formats[mydate]=%Y/%m/%d

    In your template, where you want to see the special format :

    {def $timestamp=$node.data_map.date.data_int}

    {$timestamp|l10n( 'mydate' )} : french format

    That it's.
