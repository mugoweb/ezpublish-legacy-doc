Packer

Summary

Setting to enabled or disables the use of the js / css packer.

Usage

Packer=enabled | disabled

Description

This is an optional setting which enables or disables the js / css packer. The packer is used for debugging Apache rewrite rules.Normally this function is controlled by the setting DevelopmentMode in [TemplateSettings] in site.ini for convenience, but can additionally be specifically controlled by this Packer setting. The Packer setting here trumps the DevelopmentMode setting in site.ini. By default this setting is disabled.

Two ways exist to use this setting. First it can be enabled or disabled. Secondly, this setting can be used to force packer level. This is done by setting an integer from 0 to 3 instead of the values enabled | disabled.

This setting can be set per siteaccess, but is disabled by default. This setting is case sensitive.

Examples

Example 1

Packer=disabled

This will disable the js /css packer function.

Example 2

Packer=1

This will set the packer level to 1.
