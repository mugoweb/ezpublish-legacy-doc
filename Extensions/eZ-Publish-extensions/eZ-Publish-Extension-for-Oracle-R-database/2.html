2.0

The purpose of this manual is to introduce and describe the eZ Publish Extension for OracleÂ® Database (also known as the "ezoracle" extension or "database extension") version 2.0, which makes it possible to use Oracle as a database server for eZ Publish 4.1. The sections contained within this manual will help you to understand the following issues:

  • The purpose of the "ezdb" library
  • When the database extension is needed and when it isn't
  • How to create a new eZ Publish site running on an Oracle database
  • How to migrate an existing eZ Publish site from MySQL to Oracle

Important note

People previously unfamiliar with Oracle should read the "Oracle specific notes" section first.
