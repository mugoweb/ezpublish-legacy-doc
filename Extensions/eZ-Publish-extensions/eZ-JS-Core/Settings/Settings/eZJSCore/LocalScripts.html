LocalScripts

Summary

This setting maps the local JavaScript libraries to their location.

Usage

LocalScripts[<library>]=<URL>

Description

This setting lists the local JavaScript libraries and maps them to their location. It is optional to start with a "/" to signal that the library is in the root or design folder and not in the javascript sub folder. This setting needs to be set when the setting LoadFromCDN is set to disabled.

This setting is case sensitive.

Examples

Example
[ezjscore]
LoadFromCDN=disabled
LocalScripts[yui3]=/lib/yui/3.1.1/build/yui/yui-min.js
LocalScripts[yui2]=/lib/yui/2.8.1/build/yuiloader/yuiloader-min.js
LocalScripts[jquery]=jquery-1.4.2.min.js

The system is not permitted to load from external servers. The location of the internal JavaScript libraries to use are listed under the setting LocalScripts.
