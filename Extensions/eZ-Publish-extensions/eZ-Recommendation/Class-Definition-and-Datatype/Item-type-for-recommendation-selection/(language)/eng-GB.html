Item type (for recommendation) selection

Item type (for recommendation) selection

The Item type (for recommendation) is similar to a class ID in eZ Publish. It defines a number corresponding to a given type of content. If you attribute the number “2” as Item type for recommendation to multiple classes e.g. article (main-page) and article (sub-page) then both classes are treated as one type by the recommendation service.

When the extension tracks the user activities it will tell the eZ Recommendation service that all articles are type “2” and should be treated in the same way. And when you ask for recommendations of the same type you will get node IDs from both classes as an answer.

We recommend the use of the following list to choose a suitable number Item type (for recommendation). But it is not required to strictly follow this list:
