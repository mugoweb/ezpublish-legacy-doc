Index Time Boosting

You have the option to specify boost factors during indexing. When the back-end calculates the relevancy scores, the boost factors are taken into account. This can be done in the [IndexBoost] section in your ezfind.ini. More information about the configuration of this feature can be found in the comments in the ezfind.ini file.
