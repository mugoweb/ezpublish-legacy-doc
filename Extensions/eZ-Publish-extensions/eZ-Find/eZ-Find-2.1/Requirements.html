Requirements

  • eZ Publish 4.x
    • Certified for eZ Publish 4.2
  • Java Runtime Environment:
    • for Linux: JRE 5
    • for Mac OS, Windows and Solaris: JRE 6
  • PHP Curl extension is recommended and needed for support
