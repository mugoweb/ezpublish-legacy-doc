Wiki page

Information is a key element in any project, and the Wiki page enables users to add and find information resources quickly. This feature provides a knowledge base where team members can collect, comment, combine and use information pertaining to the project. The functionality is set up with an open, editable structure, so that team members can influence the structure of the knowledge base.
