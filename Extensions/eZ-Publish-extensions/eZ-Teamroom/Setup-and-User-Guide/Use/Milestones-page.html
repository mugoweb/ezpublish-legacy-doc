Milestones page

A milestone is a collection of related tasks with the same end date. All teamroom milestones are listed on the Milestones page. To add new milestones, use the milestones button. As you will notice in the following screenshot, milestone dates that have past are highlighted in orange. Using the Lightbox link (1) enables users to add milestones to the current lightbox.

Use the Details icon to view the assigned tasks of that milestone and the Edit icon to make alterations. Adding tasks or events to a milestone can be done from the tasks or events page, not the milestones page. When a new task or event is created or edited, it is possible to add it to a milestone by selecting the proper milestone from the milestone drop down list in the milestone section, as shown in the following screenshot.

 

When a milestone has been reached users are able to close it with the Close icon. Once closed the icon will be displayed in green as shown here.

Using the links provided in the Sort by and Categories menu, users can sort the milestones or filter by category or keywords. Clicking on these links twice will revert the sorting order. Use the tags in the Keyword menu to view milestones with a particular tag. Use the Keep me updated button to subscribe to notifications.
