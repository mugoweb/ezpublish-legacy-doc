Blog page

All users can post a blog entry in any teamroom they have access to. From the teamroom frontpage or Sitemap users can view blog entries or create new posts by using the Add new entry button. To comment on an existing entry click on its title. This will display the full article where users are able to write comments, as shown in the screenshot below. Use the Edit icon (1) to edit and the Trash icon (2) to remove entries. Using the tag links provided in the Keywords menu, users can filter blogs in order to view blogs with a particular tag. Use the Keep me updated button to subscribe to notifications by email when new entries are posted. Using the Lightbox link (3) enables users to add a blog entry to the current lightbox.
