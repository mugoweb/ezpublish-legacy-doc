Keyboard shortcuts

Below, please find the keyboard shortcuts for the eZ Online Editor:

Command executed

ctrl+z

Undo

ctrl+y

Bold

ctrl+i

Italic

ctrl+u

Underline

ctrl+1-6

h1-h6

Note: On a Mac please use the "cmd" key instead of the "ctrl" key.
