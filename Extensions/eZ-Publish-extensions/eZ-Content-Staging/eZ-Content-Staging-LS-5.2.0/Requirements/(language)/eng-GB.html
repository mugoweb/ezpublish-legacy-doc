Requirements

Source Server:

  • eZ Publish >= 5.2
  • ezjscore (if using ezwt)

Target Server:

  • eZ Publish >= 5.2
  • ezprestapiprovider extension (for rest controller classes)

Requirements for working with the Website Interface (such as supported browser versions) are the same as those for the Administration Interface. The Online Editor (present in both the Website Interface and the Administration Interface) has a more specific set of requirements.

To ensure that your web server meets the proper requirements visit the eZ Publish installation manual.
