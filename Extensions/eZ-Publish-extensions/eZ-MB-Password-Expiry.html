eZ MB Password Expiry

The eZ MB Password Expiry 1.2 extension allows you to define how long passwords should remain valid. Once a password expires the extension will ask the user to create a new password. This extension is typically used on intranets and extranets and can be downloaded at the project page http://projects.ez.no/mbpaex.

This extension was developed by Microblau with eZ Systems support.
